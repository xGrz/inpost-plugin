# Artisan commands

## Publish configuration

If you want to publish a configuration file, use:

```bash
php artisan inpost:publish-config
```

## Show configuration

For printing the configuration, use:

```bash
php artisan inpost:config
```

## Publish migrations

Publish migrations for the package:
```bash
php artisan inpost:publish-migrations
```

## Points synchronization

```bash
php artisan inpost:points
```
